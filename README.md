# Java

## Github

Clone the code

```shell
git clone https://github.com/SelinaWang102/Java.git
cd Java
```

Check status

```shell
git status
```

Add new files, for example

```shell
git add README.md
```

Commit

```shell
git commit -m "some message you want to add"
```

Push the commit to the Github

```shell
git push origin master
```

Check history

```shell
git log
```
