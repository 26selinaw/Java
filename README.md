# Java

## Github

Change to desktop

```shell
cd Desktop
```

Clone the code

```shell
git clone https://github.com/26selinaw/Java.git
cd Java
```

Check status

```shell
git status
```

Add new files, for example

```shell
git add README.md
```

Commit

```shell
git commit -m "some message you want to add"
```

Push the commit to the Github

```shell
git push origin master
```

Check history

```shell
git log
```
